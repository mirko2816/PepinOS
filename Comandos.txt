Instalación de herramientas de desarrollo GCC
    apt install build-essential

Instalación del compilador de ensamblador
    apt install nasm

Instalación de QEMU
    apt install qemu-system

Compilar con NASM (NO OLVIDAR COMPILAR TAMBIEN EL KERNEL XD)
    nasm -f bin -o bootsect bootsect.asm
    nasm -f bin -o kernel kernel.asm

Concatenar el binario con ceros Y EL KERNEL y montar en el floppyA
    cat bootsect kernel /dev/zero | dd of=floppyA bs=512 count=2880

Ejecutar QEMU con el disco 
    qemu-system-i386 -boot a -fda floppyA



Comandos modificados para el cap V:
    compilar screen.c en modo 32 bits
        gcc -m32 -c screen.c -o screen.o
        nasm -f elf -o kernel.o kernel.asm

    enlazar 
        ld -m elf_i386 --oformat binary -Ttext 0x1000 kernel.o screen.o -o kernel
